## Formatos disponíveis

Identifique o formato a partir do argumento:

- **carrossel** — sequência de slides para Instagram ou LinkedIn
- **post** — publicação única (imagem + legenda)
- **reels/shorts** — script para vídeo vertical curto (15-60s)
- **legenda** — texto para acompanhar imagem já existente
- **thread** — sequência de posts conectados

Se não especificado: gere um **carrossel** (formato de maior engajamento orgânico).

## CARROSSEL — Estrutura profissional

### Anatomia de um carrossel viral

```
Slide 1: HOOK — para o scroll
Slides 2-7: CONTEÚDO — entrega o prometido
Slide final: CTA — diz o que fazer agora
```

### Slide 1 — Hook (mais importante)

O hook deve fazer a pessoa parar o scroll. Tipos mais eficazes:

**Promessa específica:**
> "5 erros que sabotam sua produtividade (e como corrigir hoje)"

**Provocação/tensão:**
> "Você está usando IA errado. Aqui está o que ninguém te conta."

**Número + benefício:**
> "3 skills que transformaram como eu crio sites em 2026"

**Pergunta que incomoda:**
> "Por que seu conteúdo não engaja mesmo sendo bom?"

**Regras do hook:**
- Máximo 8-10 palavras
- Benefício ou curiosidade clara
- Sem pontuação excessiva (!!! ou ???)
- Fonte grande, legível em miniatura

### Slides de conteúdo (2-7)

Cada slide = **uma ideia** completa:
- Título do ponto (3-5 palavras)
- Explicação (2-4 linhas máximo)
- Exemplo concreto ou dado quando possível

**Progressão lógica:**
- Problema → causa → solução
- Lista numerada (cada slide = um item)
- Antes/depois
- Mito vs. verdade
- Passo a passo

### Slide final — CTA

Diga exatamente o que fazer:
- "Salva esse carrossel para usar depois"
- "Compartilha com quem precisa ver isso"
- "Comenta qual desses você vai aplicar primeiro"
- "Segue para mais conteúdo sobre [tema]"

**Nunca:** "Gostou? Deixa seu like!" (fraco, pedinte)
**Sempre:** ação específica com motivo claro

## REELS/SHORTS — Script vertical

### Estrutura dos primeiros 3 segundos (decisivos)

```
0-1s: Visual impactante ou frase que cria tensão
1-3s: Promessa do que vai aprender/ver
```

### Corpo (4-45s)

- Uma ideia por corte (cada 3-7 segundos)
- Linguagem falada, não escrita
- Frases curtas. Pausas. Ritmo.
- Texto na tela reforça (não repete) a fala

### Encerramento (últimos 5-10s)

- Resumo em 1 frase
- CTA direto: "Segue para ver o próximo"

## POST ÚNICO — Legenda estruturada

```
[HOOK — primeira linha visível antes do "ver mais"]
[linha em branco]
[Desenvolvimento — 3-5 parágrafos curtos]
[linha em branco]
[CTA + hashtags]
```

**Primeira linha:** deve funcionar como headline de jornal — quem leu só isso já captou o valor.

---

## Instruções de output

Para carrossel, entregue:
1. **Conteúdo de cada slide** (título + texto)
2. **Sugestão de visual** para cada slide (cor dominante, tipo de imagem ou elemento)
3. **Legenda completa** para acompanhar a publicação
4. **5 sugestões de hashtags** relevantes

Para reels/shorts, entregue:
1. **Script completo** com marcações de tempo e corte
2. **Texto on-screen** para cada cena
3. **Sugestão de música** por mood (não por nome — descreva o ritmo/energia)

## Voz e tom

A menos que o argumento especifique:
- Tom: direto, sem rodeios, sem "no mundo de hoje em dia"
- Pessoa: 2ª pessoa ("você") — fala com o leitor, não sobre ele
- Sem: jargão corporativo, frases de autoajuda, excesso de emoji
- Com: exemplos reais, números quando possível, linguagem conversacional
